DomainTool
==========

by Manfred Touron

A blazing fast domain names finder.

Requirements
------------

- NodeJS
- Coffee-Script
- [Tapas](https://github.com/moul/tapas) package
- [WhoisJS](https://github.com/localtoast/whoisjs)

How to use
----------

```bash
npm install tapas whoisjs
coffee app.coffee
```
